Шаблон цього проекту створений для студентів курсу Unity 2D задля виконання завдань по MakerTime.

Проект був створений на версії 2019.4.40f1

Проект містить базову структуру папок, яка використовувалась у навчальних матеріалах протягом року, а також заздалегіть створені об'єкти, по типу Canvas (з базовим налаштуванням) та інше.

Також, додатково розширений базовий список тегів, але його можна доповнити потрібними тегами за необхідності.

У теці Sprites присутні 4 набори різних спрайтів, які ви можете використовувати для створення своєї гри. 

Усю необхідну інформацію по розробці ігор можна знайти у роздаткових матеріалах.

Номери тем, які можуть знадобитись:

Взаємодія об'єктів: №4
Створення об'єктів: №4, №7
Робота з анімаціями: №9, №19
